# datasets

Machine Learning で利用可能なデータセットなどダウンロードのサンプルスクリプトを記載します。
1 ファイルで 1 つのデータセットをダウンロードします。

スクリプト毎の簡易説明は下記に記載しますが、詳細は各スクリプトを参照してください。
