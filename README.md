## テスト実行

各問題に対する解答と実際に動かすコードは別にしています。
動作確認はテストコードを用いて行うため、下記のようにして実行します。

* 全件テスト

  ```sh
  $ make test
  # or
  $ python -m unittest discover
  ```
* 特定のテスト

  ```sh
  $ python -m unittest problem001/test_main.py
  ```
